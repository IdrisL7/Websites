Spotify Weekly Analytics & Growth Report Automation

An automated reporting system that converts messy Spotify for Artists CSV exports into a clear weekly growth narrative for artists and labels. Built with Node.js scripts to ingest and normalize CSV + Spotify Web API data into a structured dataset, then generate track-level insights and actionable next steps (what to push, where growth is coming from, and what to do next week).

Outcome: Reduced weekly reporting time from 3 hours of manual spreadsheet work to a single automated run. Runs unattended with built-in error handling and scheduled delivery.
